# Etch-a-Sketch
This is a project from The Odin Project Foundations course.

## Objectives
The primary goals of this project were to:
- Create a grid dynamically by manipulating the DOM
- Practice hover operations and event listeners
- Apply CSS Flexbox for layout

## How to Play
- Move your mouse over the grid squares to change their color
- Reset or resize the grid using the provided controls

## Built With
- HTML
- CSS (Flexbox)
- JavaScript (DOM manipulation, event handling)
